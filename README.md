# Monolith: Web Routing

### Installation

In composer.json
```json
{
  "require": {
    "monolith/web-routing": "dev-master",
    "monolith/collections": "dev-master",
    "monolith/http": "dev-master"
  },
  "repositories": [
    {
      "type": "vcs",
      "url":  "git@github.com:mitchellvanw/monolith-routing.git"
    }
  ]
}
```
